# ClaudeMod

**[claudemod.com](https://www.claudemod.com)** — a mod discovery platform for [Claude Code](https://claude.ai/code). Browse, vote on, and submit community-built configurations: CLAUDE.md setups, slash commands, hooks, and MCP server configs. Think Nexus Mods but for your AI coding environment.

Built with Next.js (App Router), Supabase, Tailwind CSS, and shadcn/ui. Deployed on Vercel.
